Complaint Box Portal

Project Overview: The Complaint Box Portal is a web-based application designed to streamline the process of submitting and resolving complaints in institutions such as schools or offices. Users can submit complaints, track their status, and receive updates from administrators in real-time. This project focuses on creating a user-friendly interface with seamless backend integration.

Key Features

  • User registration and login for students, teachers, and administrators.
  • Complaint submission form with categories, urgency level, and file uploads for supporting documents.
  • Real-time complaint status updates: Pending, In Progress, Resolved.
  • Admin dashboard to manage complaints and users, with full control over system data.
  • Interactive UI with smooth transitions and animations to enhance user experience.
